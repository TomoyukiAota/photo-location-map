Photo Location Map
Display the locations where photos were taken on a map.
Download for Windows

Photos taken with smartphones have location data (latitude and longitude).
This application fetches the location data from photos and displays the locations on a map.

Supported file formats:
  • JPEG (.jpeg, .jpg, .jpe, .jfif, .jfi, .jif)
  • HEIF (.heif, .heic)
The source code of Photo Location Map is hosted on GitHub.
Photo Location Map is developed by Tomoyuki Aota.
