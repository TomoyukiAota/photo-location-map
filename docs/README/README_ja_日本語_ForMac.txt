○ 概 要

  Photo Location Map for macOSをダウンロードしていただきありがとうございます。

  このアプリは、写真を保存したフォルダを指定すると、写真の撮影位置を地図上に表示するアプリです。

  例えば、旅行中の写真を保存したフォルダを指定すると、写真の撮影位置や日時から、
  旅の思い出を振り返ることができます。

  動作する様子については、次の動画をご覧ください。
  https://www.youtube.com/watch?v=n7hv_h1llKo

  【対応フォーマット】
  ・JPEG (.jpeg, .jpg, .jpe, .jfif, .jfi, .jif)
  ・HEIF (.heif, .heic)

  【詳細】
  スマートフォンで撮影した写真には、撮影位置が記録されています。
  撮影時に、GPSによって得られた緯度・経度の情報が写真に付与されるためです。
  このアプリは、そのような写真について、 地図上に撮影位置のピンを表示します。
  ピンをクリックすると、写真や撮影日時が表示されます。

○ イ ン ス ト ー ル 方 法

  ダウンロードしたzipファイルを展開するとdmgファイルが作成されますので、それをダブルクリックしてください。
  Photo Location Map.appを、ドラッグ＆ドロップでApplicationsフォルダにコピーしてください。

--------------------------------------------------------------

○ ア ン イ ン ス ト ー ル 方 法

  Macintosh HD -> 「アプリケーション」フォルダ内にあるPhoto Location Map.appを削除してください。
